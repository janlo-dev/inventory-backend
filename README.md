# Sistema de Control de Inventario (Backend)

Proyecto desarrollado para la gestión de stocks, enfocado en una arquitectura limpia y escalable utilizando el ecosistema de Spring.

## ⚡ Tecnologías Utilizadas
* **Java 17**
* **Spring Boot 3**
* **Spring Data JPA** para la persistencia.
* **MySQL** como base de datos.
* **Maven** para la gestión de dependencias.

## 🚀 Estado del Proyecto
Actualmente el **Core Backend (API REST)** está funcional, permitiendo la gestión completa de productos y categorías. 
*Pendiente:* Integración final con el frontend en Angular 16 (en desarrollo).

## 🛠️ Cómo ejecutarlo
1. Clonar el repositorio.
2. Configurar la base de datos en `application.properties`.
3. Ejecutar `./mvnw spring-boot:run`.
